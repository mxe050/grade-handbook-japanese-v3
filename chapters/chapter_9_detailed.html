Chapter 9：不精確さ（Imprecision）

🎯 Core Concept

不精確さ（Imprecision）は、効果推定値の信頼区間が広く、臨床的に重要な閾値をまたぐ場合にグレードダウンの対象となるドメインです。Core GRADEは、従来のOIS（Optimal Information Size）アプローチを廃止し、MID（最小重要差）に基づく閾値アプローチを導入しました。これはCore GRADEにおける最も重要な革新の一つです。[1]

従来のアプローチ vs Core GRADEアプローチ

従来のGRADE（OISアプローチ）
  • 🤔 基本的な考え方: 十分な情報量（サンプルサイズ）があるか
  • 📏 閾値の設定: RR 0.75 / 1.25（任意の基準）
  • CONTEXT 文脈化: 大・中・小の効果という文脈
  • 0️⃣ null効果の扱い: CIがnullをまたぐかどうか
Core GRADE（閾値アプローチ）
  • 🎯 基本的な考え方: 信頼区間がMID閾値をまたぐか
  • 📈 閾値の設定: MIDに基づく臨床的に意味のある閾値
  • CONTEXT 文脈化: 廃止。閾値のみで判断
  • 0️⃣ null効果の扱い: CIがMID閾値をまたぐかどうか（nullは閾値ではない）
アプローチの核心的変更

Core GRADEでは、不精確さの評価軸が「サンプルサイズは十分か？」から「効果の推定値は、臨床的に意味のある差（MID）の閾値をまたいでいるか？」へと根本的にシフトしました。これにより、より文脈に即した、臨床的に意味のある判断が可能になります。

MID閾値に基づく不精確さの評価プロセス

1
MIDの特定

各アウトカムについて、利益のためのMIDと害のためのMIDを事前に特定します。MIDには不確実性が伴うため、「もっともらしいMIDの範囲」として幅を持たせることが推奨されます。

➔
2
閾値の設定

MIDに基づき、効果の大きさを判断する閾値を設定します。Core GRADEでは、null効果は閾値ではありません。閾値は臨床的に重要な利益または害の境界を示します。

➔
3
信頼区間と閾値の比較

95%信頼区間（CI）が設定したMID閾値をまたぐかを確認します。CIが閾値をまたぐ場合、不精確さによるグレードダウンを検討します。

不精確さの判断基準 ― 4つのシナリオ

シナリオ1：グレードダウンなし

CIの上限と下限の両方がMID閾値の同じ側にあり、閾値をまたがない。効果の方向性と大きさについて十分な精度があると判断します。

MID
シナリオ2：1段階グレードダウン

CIが1つのMID閾値をまたぐ。効果が臨床的に重要かどうかについて不確実性があるため、1段階グレードダウンします。

MID
シナリオ3：2段階グレードダウン

CIが利益と害の両方のMID閾値をまたぐ。効果が利益をもたらすか害をもたらすかさえ不確実なため、2段階グレードダウンします。

MID(益)
MID(害)
シナリオ4：null効果に近い場合

点推定値がnull効果に近く、CIが両方のMID閾値の内側にある場合、「ほとんど効果がない」と結論付け、グレードダウンしません。

MID(益)
MID(害)

アウトカムの種類と不精確さ

二値アウトカム

相対効果（RR）と絶対効果（RD）の両方で評価しますが、特に絶対効果の不精確さが推奨策定において最も重要です。

  • 📊 ベースラインリスクが異なると、同じRRでも絶対効果の不精確さは変わります。
連続アウトカム

平均差（MD）または標準化平均差（SMD）の信頼区間とMID閾値を比較します。

  • 📏 SMDの場合、一般的に0.5 SDがMIDの目安とされますが、アウトカム固有のMIDを優先します。

不精確さと不一致性の関係

⚠️ 二重カウントの回避

不精確さと不一致性は密接に関連しています。異質性が大きいことで信頼区間が広がっている場合、その問題は主に「不一致性」のドメインで評価し、不精確さで再度グレードダウンする二重カウントを避けるべきです。[2]

具体例：COVID-19患者に対するニルマトレルビルの不精確さ評価

💡 具体例

アウトカム：入院または死亡（二値アウトカム）
効果推定値：RR = 0.57（95% CI: 0.36 - 0.89）
MID：RR = 0.75（パネル調査で設定）

判断：CIの上限（0.89）がMID（0.75）を超えているため、CIがMID閾値をまたいでいると判断し、1段階グレードダウンします。

※絶対効果で評価した場合も同様の結論に至ることが多いです。例えば、高リスク群（ベースラインリスク10%）では1000人あたり43人の入院減少（CI: 11-64）となり、MIDが25人の場合、CIの下限（11）がMIDを下回るため、やはり1段階グレードダウンとなります。

✅ Chapter 9のまとめ
  • 🎯 Core GRADEは従来のOISアプローチを廃止し、MID閾値ベースのアプローチを導入しました。
  • 🚫 null効果は閾値ではありません。臨床的に意味のあるMIDを使用します。
  • 📉 CIが閾値をまたぐ数に応じて、0, 1, または2段階のグレードダウンを行います。
  • 🔄 null効果に近い場合は、確実性評価のターゲットを「ほとんど効果がない」に修正します。
  • ⚠️ 不精確さと不一致性の二重カウントを避けることが重要です。
  • 📈 絶対効果の不精確さが、最終的な推奨の策定において最も関連性が高くなります。
